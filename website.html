What we've learned

  • setup() This is run once, at the beginning of the program, when you click the (â–º) Run button. It might look like this: 
    void setup() {
  // Set the size of canvas
  // size(width,height)
  size(480,480);
  // Fill it with a uniform gray background
  background(70,70,70);
}
  • draw() This is run many times a second (60Hz!). It lets us create animations that vary with time.
